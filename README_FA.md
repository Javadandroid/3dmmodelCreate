# راهنمای سریع ساخت مدل چرخ‌دنده/پولی (CEG7425B)

از روی عکس‌ها نمی‌شود **ابعاد دقیق** را با اطمینان استخراج کرد؛ برای قطعه‌ی مکانیکیِ درگیر با تسمه، حتماً با **کولیس** اندازه بگیرید و بعد مدل را با اعداد خودتان بسازید.

## فایل آماده
- `beko_pulley_generator.py` یک مدل ساده‌ی پولی دندانه‌دار (دندانه‌های صاف) را به **STL** تولید می‌کند.
- واحد خروجی STL: **میلی‌متر** (در اسلایسر هم روی mm بگذارید).

ساخت فایل STL:
```bash
python3 beko_pulley_generator.py --out beko_pulley.stl
```

راهنمای پارامترها:
```bash
python3 beko_pulley_generator.py --help
```

نکته: برای خاموش‌کردن مگنت‌ها `--magnet-count 0` را بزنید.
ری‌جنریت فایل پیش‌فرض پروژه (همان که در ادیتور باز می‌کنی):
```bash
python3 beko_pulley_generator.py --out beko_ceg7425b_pulley_default.stl
```
حذف/اضافه کردن فلنج بالایی:
- پیش‌فرض فلنج بالایی حذف است (`--top-flange-t 0`)
- اگر خواستی فلنج بالایی هم داشته باشد مثلاً: `--top-flange-t 1.2`

## چه اندازه‌هایی لازم است؟
این‌ها را از قطعه‌ی اصلی اندازه بگیرید:
1) `outer-d` قطر بیرونی روی نوک دندانه‌ها  
2) `tooth-width` عرض قسمت دندانه‌دار (بدون فلنج‌ها)  
3) `flange-d` قطر فلنج‌ها  
4) `flange-t` ضخامت هر فلنج  
5) `bore-d` قطر سوراخ وسط (یا اگر D-shaft/شیاردار است، شکل دقیقش)  
6) `teeth` تعداد دندانه‌ها (دور تا دور بشمارید)  
7) اگر تسمه تایم است: **گام تسمه** (فاصله‌ی مرکز تا مرکز دو دندانه روی تسمه)
8) جای مگنت‌ها (اگر دستگاه شمارش دور دارد):
   - `magnet-d` قطر مگنت
   - `magnet-depth` عمق جای مگنت (برای جا رفتن کامل)
   - `magnet-boss-d` قطر زائده نگهدارنده
   - `magnet-boss-h` ارتفاع زائده زیر قطعه
   - `magnet-center-r` فاصله مرکز زائده تا مرکز قطعه
   - `magnet-angle` زاویه‌ی شروع (برای چرخاندن محل مگنت‌ها)

نکته‌ی مهم: اگر `teeth` را اشتباه بزنید، با تسمه درست درگیر نمی‌شود.

## تلرانس پیشنهادی برای پرینت
- سوراخ (بوش): معمولاً `bore-d` را **۰.۱۵ تا ۰.۳۰ میلی‌متر بزرگ‌تر** از اندازه‌ی واقعی بگیرید (بسته به پرینتر و جنس).
- اگر قرار است فیت سفت باشد، یک تست کوچک با چند قطر مختلف چاپ کنید.

## جنس پیشنهادی
- برای داخل دستگاه قهوه‌ساز: **PETG** یا **Nylon/PA** بهتر از PLA است (حرارت و سایش).
